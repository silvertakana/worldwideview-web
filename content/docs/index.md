# WorldWideView Documentation

Welcome to the WorldWideView developer documentation. Here you'll find guides, API references, and quickstarts for building on the platform.

## What is WorldWideView?

WorldWideView is a real-time geospatial intelligence platform built on a 3D globe. It tracks aircraft, ships, signals, and any kind of geospatial data through a plugin-driven architecture — open source, extensible, and designed for developers.
