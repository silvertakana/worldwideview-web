# Using the Globe

This guide covers how to move around the globe, control which time period you are viewing, inspect live entities, and customize the visual style.

## Navigating the Globe

Use your mouse to move the camera in three dimensions.

| Action | How to do it |
|--------|--------------|
| Pan (move the map) | Click and drag |
| Zoom in / out | Scroll wheel |
| Tilt and rotate | Right-click and drag |
| Reset view | Double-click on empty globe area |

You can also hold **Ctrl** while dragging to tilt with the left mouse button, or rotate with the right.

## Jumping to a Region

The header contains a row of region buttons. Click any region name to fly the camera to that area instantly.

Available regions: **Global**, **Americas**, **Europe**, **MENA**, **Asia Pacific**, **Africa**, **Oceania**, **Arctic**.

Clicking **Global** resets the camera to the full-Earth overview.

## Controlling the Time Window

The time window selector in the header sets how far back in time the displayed data goes. Choose from:

- **1h** - Last hour only (fastest load, most focused)
- **6h** - Last six hours
- **24h** - Last 24 hours
- **48h** - Last two days
- **7d** - Last seven days (most history, may fetch more data)

Smaller windows load faster and reduce visual clutter. Larger windows let you see patterns over a longer period.

## Using the Timeline

The timeline bar runs along the bottom of the screen. It has two modes.

**Live mode (default):** The platform streams the most current data as it arrives. The playhead stays at the right edge and advances in real time. No action required to stay live.

**Playback mode:** Lets you scrub back through historical data within the selected time window.

To switch to playback mode:
1. Click the play/pause toggle on the timeline bar. The playhead detaches from the live edge.
2. Drag the scrubber left to move back in time, or use the playback controls.
3. Select a playback speed: **1x**, **2x**, **10x**, or **100x**.
4. Click the toggle again (or drag the scrubber to the far right) to return to live mode.

## Inspecting an Entity

Click any aircraft, ship, or marker on the globe to select it. A detail card opens showing that entity's live data - position, speed, heading, identification, and any additional fields provided by the data layer.

Click empty space on the globe to deselect and close the card.

Some areas show clustered entities when many items are close together. Click a cluster to expand it and see individual items.

## Switching the Map Style

The theme picker in the header lets you switch the globe's visual style.

| Theme | Appearance |
|-------|------------|
| Dark | Dark background, high contrast (default) |
| Black | Pure black background, maximum contrast |
| Light | Light background, suitable for bright environments |
| Legacy | Original color scheme from earlier versions |

Switching themes changes the color palette only - it does not affect which data is displayed or loaded.
